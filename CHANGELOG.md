# Changelog

## 1.0.1 - 2026-07-20

- Renamed the application to Audient iD Volume Control.
- Documented experimental, unverified compatibility with the iD4 MKII, iD24, and iD44 MKII.
- Changed the Windows package from a self-extracting executable to a normal ZIP folder build.
- Avoided a Microsoft Defender false positive caused by the previous PyInstaller one-file package.
- Clarified installation and custom Audient driver path instructions.

## 1.0.0 - 2026-07-20

- Direct hardware monitor-volume control for the Audient iD14 MKII.
- System-tray slider, mute control, and compact volume OSD.
- Global volume and mute hotkeys.
- Automatic device reconnection and volume restoration after USB reset or sleep.
- Configurable safe volume ceiling.
- Optional per-user Windows startup.
